## I. What is recursion?

Một hàm gọi lại chính nó bên trong body code được gọi là hàm đệ quy. Và kỹ thuật này được gọi là đệ quy.

### Syntax

```c
void recurse() {
    ... ... ...
    recurse();       // gọi lại chính nó bên trong nó
    ... ... ...
}

int main() {
    ... ... ...
    recurse();
    ... ... ...
}
```
